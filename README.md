# SP Landscaping LLC Website

Simple static site for SP Landscaping LLC.

## Run locally
```bash
py -m http.server 8000
```

Then open:
`http://localhost:8000`

## Notes
- Hero image, services images, and homepage layout are preserved.
- All project images were renamed in place with descriptive SEO-style filenames.
- Gallery page includes all project photos with lazy-loading.
